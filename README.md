# 📚 UsedBookFlow – Frontend

중고·재고 도서 매칭 서비스 **UsedBookFlow**의 프론트엔드입니다.  
React + TypeScript + Vite 기반으로 구축되었으며, 재고 등록·AI 추천·신청 기능을 제공합니다.

---

## 🚀 주요 기능
- **홈 화면**: 재고 등록 / AI 추천 / 나의 현황 진입 카드  
- **재고 도서 등록**: Stepper 형식의 5단계 입력 + 완료 모달  
- **AI 도서 추천**:  
  - 안내 문구 순차 출력 → 사용자 입력 → 추천 진행 로딩 → 결과 리스트  
  - 카드 flip 효과 (앞면: 책 정보, 뒷면: 상세정보 + 신청 버튼)  
- **로그인 / 회원가입**: 단일 폼 구성 및 스타일 통일  

---

## 🧩 기술 스택
- **Framework**: React 18 + Vite  
- **Language**: TypeScript  
- **Styling**: CSS Modules  
- **Icons**: [Lucide React](https://lucide.dev/)
